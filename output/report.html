spring-rest-sakila

A Spring Boot REST API service that provides CRUD operations on the MySQL Sakila sample database, which models a DVD rental store. The service exposes endpoints for managing actors, customers, films, payments, rentals, staffs, stores, and reporting data. It serves as a sample project demonstrating best practices for building REST APIs with Spring Boot, including support for complex queries and comprehensive API documentation.

Technology Stack: Spring Boot Spring Data JPA Spring HATEOAS Spring REST Docs Spring Security Spring Actuator QueryDSL MapStruct Lombok Blaze Persistence MySQL Connector JWT (JJWT) Prometheus Asciidoctor
The codebase is organized as a layered monolithic Spring Boot application with clear separation of concerns. It uses Spring Data JPA for data access, MapStruct for entity-to-DTO mapping, and QueryDSL for complex queries. The architecture includes DTOs for API contracts, domain entities with embeddable value objects (FullName), mappers for transformation, and utility classes for constants and common definitions (error codes, categories, ratings, languages, HAL relations). Spring HATEOAS is integrated for hypermedia support, and Spring REST Docs with Asciidoctor is used for API documentation generation.
