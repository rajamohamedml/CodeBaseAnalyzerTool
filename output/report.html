spring-rest-sakila

A REST API service that provides CRUD and complex query operations on the Sakila sample database, which models a DVD rental store company with data about films, actors, customers, rentals, and more. Built with Spring Boot, it offers comprehensive endpoints for accessing and manipulating Sakila database entities while supporting JWT authentication, advanced querying through QueryDSL, and standardized REST documentation via OpenAPI/Swagger and REST Docs.

🕐 Generated July 11, 2026 at 11:32 PM UTC
Technology Stack: Spring Boot Web Spring Data JPA Spring HATEOAS Spring Security Spring REST Docs QueryDSL MapStruct Blaze-Persistence MySQL Connector JJWT (JWT library) Lombok Spring Actuator Prometheus Asciidoctor Gradle
The application follows a layered architecture with clear separation of concerns: Controller layer handles HTTP requests, Service layer implements business logic, Repository layer provides JPA-based data access, and a DTO/Entity layer for data transfer and persistence. Key architectural features include Spring Security with JWT token-based authentication, Spring HATEOAS for hypermedia-driven REST responses, QueryDSL integration for advanced querying, MapStruct for entity-to-DTO mapping, and comprehensive exception handling through a centralized GlobalExceptionHandler. MDC logging and observability support (via Spring Actuator and Prometheus) are configured at the infrastructure level.
