spring-rest-sakila

A comprehensive REST API service built with Spring Boot that provides CRUD operations and complex queries against the Sakila sample database—a MySQL dataset modeling a DVD rental store company. The service exposes endpoints for managing films, actors, customers, rentals, payments, staff, stores, and related entities, with support for filtering, pagination, and detailed relationship queries. The application demonstrates modern Spring ecosystem practices including Spring Data JPA for persistence, Spring HATEOAS for hypermedia-driven REST responses, QueryDSL for dynamic queries, JWT-based authentication and authorization, and comprehensive API documentation via OpenAPI/Swagger and Postman collections.
